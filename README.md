# BankingSystemAssignment

Here are the main project methods/topics implemented :

open_account: Create a new account with user-entered details.

addAccount: Add the Account object to the repository/database.

deposit: Add money to a selected account.

withdraw: Remove money from a selected account, with checks for sufficient balance.

transfer_amount: Transfer funds from one account to another.

getAllAccountsInfo: Display summary/details of all accounts in the system.
